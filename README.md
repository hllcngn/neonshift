### NEONSHIFT ASCII

**Speedpainting brain interface**\
**Prototype No1**

**Supports 2 colors!**

### usage:

```
make
./neonshift
```

### key bindings:

**movement**

**g** : vertical sweeping\
**b** : reverse sweeping\
**-/+** : sweeping speed\
**spacebar** : pause\
**hjkl** : move

**painting**

**z** : paint\
**c** : switch color\
**;** : stroke/spot\
**p** : invert

**functional**

**q** : quit\
**s** : save\
/!\ Neonshift prototype saves to the 'painting' file only. If you want to keep your stuff, you need to copy that file.\
If the 'painting' file exists upon launch, it is loaded.
